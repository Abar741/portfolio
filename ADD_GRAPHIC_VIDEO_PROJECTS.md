## **🛠️ HOW TO ADD YOUR OWN PROJECTS:**

### **✅ Add Graphic Design Projects**
1. **Go to Admin Panel**: `/admin`
2. **Navigate**: Projects → Add Project
3. **Fill Details**:
   - **Title**: Your project name
   - **Category**: Select "Graphic Design"
   - **Description**: Describe your design work
   - **Technologies**: Photoshop, Illustrator, Figma, etc.
   - **Image**: Upload your design mockup
   - **Live Link**: Link to Behance, Dribbble, or portfolio

### **✅ Add Video Editing Projects**
1. **Go to Admin Panel**: `/admin`
2. **Navigate**: Projects → Add Project
3. **Fill Details**:
   - **Title**: Your video project name
   - **Category**: Select "Video Editing"
   - **Description**: Describe your video work
   - **Technologies**: Premiere Pro, After Effects, etc.
   - **Image**: Upload video thumbnail/poster
   - **Video**: Upload your video file
   - **Live Link**: Link to Vimeo, YouTube, or portfolio

## **🎯 BEST PRACTICES:**

### **✅ For Graphic Design Projects**
- **High-Quality Images**: Use PNG or JPG formats
- **Multiple Views**: Show different angles/details
- **Process Documentation**: Include sketches/iterations
- **Client Information**: Mention client if appropriate
- **Tools Used**: List all design software

### **✅ For Video Editing Projects**
- **Video Formats**: MP4 works best for web
- **Poster Images**: High-quality thumbnails
- **Video Length**: Keep under 5 minutes for portfolio
- **Compression**: Optimize for web playback
- **Hosting**: Use Vimeo, YouTube, or self-host
